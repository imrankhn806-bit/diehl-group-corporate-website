# Diehl Group – Forensic Architecture & Expert Witness Website

## Overview

Diehl Group is a professional consulting firm specializing in forensic architecture, construction consulting, expert witness testimony, building investigations, litigation support, and architectural analysis.

This project involved developing and maintaining a professional corporate website focused on presenting specialized consulting services while delivering a modern, responsive, and user-friendly experience.

**Live Website:** https://diehlgroup.com/

The goal of the project was to establish a professional online presence capable of:
